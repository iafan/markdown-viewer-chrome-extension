# Markdown Viewer

A Chrome extension that renders Markdown files in the browser.

## Tech Stack

- TypeScript
- Vite (for building/bundling the extension)
- Chrome Extension Manifest V3
- marked (for markdown parsing)

## Build

Always rebuild after making changes to the code:

```bash
npm run build 
```
